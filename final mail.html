Characteristics of the Database Approach: 1. Self-describing nature of a database system: A database stores not only the data but also information about the data (metadata), such as data types, relationships, and constraints. 2. Program-data independence: The structure of the database is separate from application programs. Changes made to the database structure do not require major changes to the programs. 3. Support for multiple views of data: Different users can view the same database in different ways according to their requirements. 4. Data sharing and multi-user transaction processing: Many users can access and update the database simultaneously while maintaining data consistency and avoiding conflicts. 5. Reduced data redundancy and improved consistency: Duplicate data is minimized, which helps maintain accurate and consistent information throughout the database.
